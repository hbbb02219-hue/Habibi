# Number History Bot

A consent-first Telegram bot to store and show moderated history entries for phone numbers.

**Important:** Do NOT commit real secrets. Use `.env` for BOT_TOKEN, API_ID, API_HASH, MONGO_URI, ADMIN_IDS.
